teamexcel_flutter

The flutter application for the Team Excel mobile application experience.

Getting Started

Install Flutter Dev Tools

Setup your environment and prefered IDE using flutter.dev:Get Started

Install AWS Amplify CLI Environment

If you want to interact with the AWS Amplify service back-end, follow these instructions. If you're just working on the front-end of the app and don't need to interact/change the application Models/Schemas, then you can skip these tools (the app will still run and connect to the existing 'dev' region Amplify services).

  1. Install the Amplify CLI Tools
    1. Note: Use your Team Excel AWS credentials to create a new AccessKey/SecretKey pair for the "Configuration" step and save your key information (contact @JELaVallee if you need an account!)
    2. Follow these instructions to install and setup your Amplify CLI configuration: Amplify CLI:Install
  2. Synchronize your local Amplify environment with the cloud environment
    1. In the root of this project, run the command amplify pull
    2. Follow the prompts to select:
      1. Authentication method: AWS access keys
      2. accessKeyId: {the access key id you created above}
      3. secretAccessKey: {the secret key you created above}
      4. region: us-east-2
      5. which app?: teamexcelmobile
      6. Select your prefered IDE and accept defaults for the remaining
    3. When the syncronization completes, you should be able to run the command amplify status and see a list of deployed services.

Start the Developing!
